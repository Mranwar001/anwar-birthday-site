# 🎂 Anwar's Birthday Site

A cinematic, premium, and romantic birthday website designed for a truly magical user experience. This project features high-end animations, smooth transitions, and immersive visuals to celebrate a special day.

## ✨ Features

- 🎭 **Cinematic Preloader**: A bespoke loading experience to set the stage.
- 🎬 **Hero Section**: A stunning entrance with smooth entrance animations.
- 📸 **Interactive Gallery**: A beautifully styled space for memories.
- 📖 **Story Timeline**: A romantic journey through time using interactive storytelling.
- ✉️ **Message Carousel**: A personalized space for heartfelt messages.
- 🎆 **Celebration Mode**: Interactive confetti and firework effects to mark the moment.
- 🎵 **Integrated Music Player**: Ambient background music control for the perfect vibe.
- 🌌 **Premium Aesthetics**: Dark mode by default, featuring glassmorphism, dynamic blur blobs, and custom micro-animations.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Effects**: [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository or download the source code.
2. Navigate to the project directory:
   ```bash
   cd anwar-birthday-site
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build

Create a production-ready build:
```bash
npm run build
```

## 📂 Project Structure

- `src/components`: UI components (Hero, Gallery, Timeline, etc.)
- `src/hooks`: Custom React hooks (Confetti, etc.)
- `src/App.tsx`: Main application entry point and layout
- `src/index.css`: Global styles and Tailwind configurations
- `dist/`: Production build output
